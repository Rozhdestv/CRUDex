# CRUDex CRUD simple en Express + Vite

CRUDex is a CRUD web application made in Express and Vite Vanilla.

## Installation

Clone the repo.

```bash
npm install requirements
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
